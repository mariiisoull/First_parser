## Задание: парсинг данных

## Цель

Написать Python-скрипт, который собирает данные с одного из сайтов:

- https://house.kg
- https://mashina.kg
- https://enter.kg

## Требования

1. Выбрать один сайт.
2. Использовать `requests` и `BeautifulSoup`.
3. Обработать несколько страниц сайта.
4. Для каждой записи сохранить основные данные:
   - название;
   - цена;
   - дополнительные характеристики;
   - ссылка.
5. Очистить данные:
   - удалить лишние пробелы;
   - преобразовать цену в число;
   - отсутствующие значения заменить на `None`;
   - удалить дубликаты.
6. Сохранить результат в файлы:
   - `result.json`;
   - `result.csv`.

> **Примечание для `house.kg`:**
>
> В качестве основного задания достаточно выбрать **один раздел** (например, *Продажа квартир*).
>
> **Дополнительное задание:** реализовать парсинг **нескольких разделов** (например, квартиры, дома и коммерческая недвижимость) с объединением данных в один итоговый файл.
